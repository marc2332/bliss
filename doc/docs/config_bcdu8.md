# BCDU8

## Description

This module allows to control Bunch Clock Delay Unit (BCDU8) by providing access to BCDU8 commands


## Configuration

### YAML configuration file example


```YAML
    plugin: bliss
    class: Bcdu8
    name: bcdu8id19
    tcp:
      url: bcdu8id191.esrf.fr:5001
```
